# Whisky Culinary Vault

📌 Project Overview
The Whisky Vault is a Django-powered interactive platform dedicated to culinary enthusiasts. Built on the belief that "home meals are just as good, if not better," this platform serves as a comprehensive repository for users to explore, share, and celebrate the art of cooking.

Whether you are looking for a quick breakfast or a detailed holiday feast, the Vault provides a community-driven space to preserve and discover world-class recipes.

🚀 Key Features & Requirements
1. Navigation & Discovery
The platform is organized into three core categories to ensure easy access to content:

Daily Recipes: Includes dropdown options for Breakfast, Lunch, and Dinner.

Health & Diet: Dedicated recipes for specific nutritional needs.

Holidays: Special occasion meals and festive traditions.

2. User Interaction
Authentication: Secure Sign-up, Login, and Logout system for all users.

Recipe Submission: Logged-in users can contribute their own recipes via a custom submission form (includes Ingredients, Instructions, and Image uploads).

Search Functionality: A dynamic search bar allows users to find recipes by keywords or ingredients instantly.

3. Visuals & Engagement
Chef’s Special: A featured video section on the landing page showcasing expert culinary skills.

Community Feedback: A dedicated testimonial section highlighting reviews from at least 3 community members.

🛠️ Technical Setup & Installation
Prerequisites
Python 3.8 or higher

Django 4.2+

Pillow (for Image handling)

Installation Steps
Clone the Project:

Bash
git clone https://github.com/your-username/whisky-vault.git
cd whisky-vault
Install Dependencies:

Bash
pip install -r requirements.txt
Database Migrations:

Bash
py manage.py makemigrations
py manage.py migrate
Launch the Vault:

Bash
py manage.py runserver
Access the site at http://127.0.0.1:8000/
